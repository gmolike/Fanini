# Shared Components

Die Shared Layer enthält alle wiederverwendbaren UI-Komponenten der Faninitiative Spandau.

## Kategorien

### 🎨 Animated

- AnimatedText - Fade-in Text Animationen
- AnimatedNumber - Animierte Zahlenänderungen

### 🔘 Buttons

- Button - Haupt-Button-Komponente mit Varianten
- ButtonGroup - Gruppierte Buttons

### 📊 Display

- EmailDisplay - E-Mail Anzeige mit Link
- PhoneDisplay - Telefonnummer mit Link
- DateDisplay - Formatierte Datumsanzeige
- BooleanDisplay - Ja/Nein Anzeige

### 🏷️ Enum

- EnumBadge - Type-safe Badge für Enums
- EnumLabel - Type-safe Label für Enums

### 📈 Charts

- OrgChart - Organigramm Komponente
- WorkflowChart - Workflow Visualisierung

### 🎯 Feedback

- LoadingState - Loading/Error/Empty States
- ErrorBoundary - Fehlerbehandlung

### 🌟 Effects

- FloatingCard - 3D Hover Effekt
- GlassCard - Glassmorphism
- ParallaxCard - Parallax Scroll

### 🧩 Layout

- Container - Responsive Container
- PageHeader - Seiten-Header
- PageSection - Seiten-Abschnitte

### 🔄 Navigation

- ModernTabs - Tab Navigation
- Breadcrumb - Breadcrumb Navigation
